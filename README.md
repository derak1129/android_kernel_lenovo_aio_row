Daredevil Kernel for Lenovo A7000-a
==============

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | 1.5GHz 64-bit Octa-Core MT6752
GPU     | Mali-T760MP2
Memory  | 2GB RAM
Android Version | 6.0
Storage | 8GB
Display | 5.5" IPS 1280 x 720 px
Camera  | 8MPx, LED Flash

This branch is for building VIBE MM/7.x based ROMs kernel.

### Compilation
        
        $ cd daredevil-kernel

        $ sh daredevil.sh
